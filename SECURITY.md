# Security Policy

Please report suspected vulnerabilities privately.

## How To Report

Email `admin@vocoder.app` with:

- A short description of the issue
- The affected package and version
- Reproduction steps or a proof of concept
- Any impact or exploitability notes

Please do not open a public GitHub issue for security-sensitive reports.

## What To Expect

- We will aim to acknowledge new reports within 3 business days.
- We may ask follow-up questions or request a private reproduction.
- Once a fix is available, we will coordinate disclosure and release notes.

## Scope

This policy applies to the public SDK packages in this repository, including:

- `@vocoder/react`
- `@vocoder/cli`
- `@vocoder/mcp`
- `@vocoder/plugin`
- `@vocoder/core`
- `@vocoder/config`
- `@vocoder/extractor`
