# Code of Conduct

We want the Vocoder SDK community to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but be open to feedback.
- Focus on the code, docs, and behavior being discussed.
- Share context clearly, especially when reporting bugs or regressions.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Doxxing or sharing private information
- Threats, intimidation, or abusive language
- Disruptive behavior that makes collaboration harder for others

## Enforcement

Project maintainers may edit, remove, or reject content that violates this code of conduct.

If you need to report an issue, email `admin@vocoder.app`.
